<h3>Radio Group</h3>

| 属性                     | 描述       | 类型              | 推荐值 | 默认    |
| ------------------------ | ---------- | ----------------- | ------ | ------- |
| **modelValue / v-model** | 绑定值     | `string / number` | -      | -       |

<h3>Radio</h3>

| 属性                     | 描述               | 类型              | 推荐值 | 默认    |
| ------------------------ | ------------------ | ----------------- | ------ | ------- |
| **modelValue / v-model** | 绑定值             | `string / number` | -      | -       |
| **label**                | 用于显示的文本     | `string`          | -      | -       |
| **disabled**             | 禁用此选项         | `boolean`         | -      | `false` |
| **description**          | 用于显示的详情描述 | `string`          | -      | -       |
