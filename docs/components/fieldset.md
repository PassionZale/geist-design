<ex-code name="ex-fieldset-disabled">

添加 <g-code>disabled</g-code> 属性会影响所有子组件。

</ex-code>

<ex-footer>

| 属性         | 描述           | 类型      | 推荐值           | 默认    |
| ------------ | -------------- | --------- | ---------------- | ------- |
| **footer**   | 组件的底部内容 | `string`  | -                | -       |
| **disabled** | 是否禁用       | `boolean` | `true` / `false` | `false` |

</ex-footer>
