# 描述 Description

<ex-code name="ex-description-basic">

组件 <g-code>Description</g-code> 包含标题与内容。

</ex-code>

<ex-code name="ex-description-slot">

同样支持 <g-code>slot</g-code> 模式。

</ex-code>

<ex-footer>

| 属性        | 描述 | 类型     | 推荐值 | 默认 |
| ----------- | ---- | -------- | ------ | ---- |
| **title**   | 标题 | `string` | -      | -    |
| **content** | 内容 | `string` | -      | -    |

</ex-footer>
