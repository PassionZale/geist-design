# Display / 陈列框

<ex-code name="ex-display-basic">

默认效果

</ex-code>

<ex-code name="ex-display-width">

为陈列框中的内容设置合适的宽度。

</ex-code>

<ex-code name="ex-display-caption">

使用 <g-code>slot</g-code> 添加自定义的说明内容。

</ex-code>

<ex-footer>

| 属性        | 描述       | 类型     | 推荐值 | 默认    |
| ----------- | ---------- | -------- | ------ | ------- |
| **width**   | 内容的宽度 | `string` | -      | -       |
| **caption** | 说明文字   | `string` | -      | -       |
| **shadow**  | 阴影效果   | `string` | -      | `false` |

</ex-footer>
